Following Functionaity are Covered as part of SBA1:

User Module:

	a. New User Registration
	b. Existing User Login and Validation
	c. Apply For new Loan
	d. Edit Existing Loan Request if not Approved/Rejected
	e. Track Loan Status for the Logged IN User
	f. Apply New Loan Functionality
	
Admin Module:

	a. Admin User Validation
	b. Process The existing Loan Request which is not Approved/Rejected
	c. Calculation of Monthly payment, Loan closure Date and Term payment amount and updation of Status(Approve/Rejected)
	d. Listing all the Loan Request irrespctive of Status.
